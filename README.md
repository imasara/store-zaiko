# 스토어 재고 보기

`brand.naver.com` 및 `smartstore.naver.com` 상품 목록 페이지에서 재고를 확인할 수 있도록 도와주는 Chrome Extension (Manifest V3)입니다.

이 확장 프로그램은 페이지의 `window.__PRELOADED_STATE__` 데이터를 읽어 다음 기능을 제공합니다.

- 우클릭 메뉴 `상품정보 모아보기`
- 우클릭 메뉴 `재고 확인하기`
- 상품 카드 가격 옆에 `재고:xxx개` 자동 표시
- 상품 정보를 썸네일 기반 테이블 팝업으로 표시

## 설치 방법

1. Chrome에서 `chrome://extensions` 를 엽니다.
2. 우측 상단의 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다`를 클릭합니다.
4. 이 폴더를 선택합니다.

## 지원 페이지

- `https://brand.naver.com/*`
- `https://smartstore.naver.com/*`

## 기능 설명

### 1. 상품정보 모아보기

우클릭 메뉴에서 `상품정보 모아보기`를 선택하면 현재 페이지의 상품 목록을 팝업 테이블로 보여줍니다.

- 썸네일
- 상품명
- 판매가
- 재고 수량

재고가 `0`인 경우에는 해당 행을 회색으로 표시하고, 재고 칸에는 빨간색 `품절` 문구를 표시합니다.

썸네일을 클릭하면 현재 스토어 기준 상품 상세 페이지로 이동합니다.

### 2. 재고 확인하기

우클릭 메뉴에서 `재고 확인하기`를 선택하면 현재 페이지의 상품 카드에 재고 수량 표시를 다시 적용합니다.

### 3. 자동 재고 표시

페이지 진입 후 약 3초 뒤 자동으로 재고 표시를 시도합니다. React 렌더링이 끝난 뒤 실행되도록 약간 지연시켰습니다.

상품 카드에서 아래와 같은 형태로 재고를 표시합니다.

- `재고:91개`
- `재고:0개`

## 참고

- 데이터가 없는 페이지에서는 기능이 동작하지 않을 수 있습니다.
- 확장 프로그램을 수정한 뒤에는 `chrome://extensions` 에서 새로고침해야 변경 사항이 반영됩니다.
- 이미 열어 둔 탭도 새로고침하면 가장 안정적으로 동작합니다.

## 면책 조항

- 이 확장 프로그램은 스토어 페이지에 이미 로드된 데이터를 바탕으로 재고 정보를 표시하는 비공식 도구입니다.
- 표시되는 재고 수량은 실제 판매 시점의 재고와 차이가 있을 수 있으며, 주문 가능 여부를 보장하지 않습니다.
- 스토어의 페이지 구조, 데이터 형식, 서비스 정책 변경에 따라 일부 또는 전체 기능이 예고 없이 동작하지 않을 수 있습니다.
- 본 확장 프로그램 사용으로 인해 발생하는 직접적 또는 간접적 손해에 대해 제작자는 책임을 지지 않습니다.
